# LearningPlaywright

Playwright learning project. This repository contains basic examples under `Chapter_01_Basics`.

Remote: https://github.com/sonalibagwe/LearningPlaywright.git

To push locally:

```powershell
git init
git add .
git -c user.email="you@example.com" -c user.name="Your Name" commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/sonalibagwe/LearningPlaywright.git
git push -u origin main
```
